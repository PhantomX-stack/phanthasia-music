# Phantasia Music

A simplified Android Compose app with a flat, easy-to-navigate source layout.

## Structure
```
app/
  src/main/
    kotlin/com/phantasia/music/
      AppNavigation.kt
      AppScreens.kt
      Data.kt
      MainActivity.kt
      Theme.kt
    res/
      values/themes.xml
      xml/network_security_config.xml
    AndroidManifest.xml
  build.gradle.kts
build.gradle.kts
libs.versions.toml
settings.gradle.kts
```

## Build
```bash
./gradlew assembleDebug
```
